# sofic

> Check the source files in directories

Sofic: **So**urce **fi**le **c**hecker.

## Installation

```sh
npm install --global sofic
```

## Usage

```
$ sofic --help

  Usage
    $ sofic
    $ sofic add <tool>
    $ sofic check <path>

  Examples
    $ sofic
    $ sofic add eslint
    $ sofic check C:\Users\knut\dev\sofic
    $ sofic check C:\Users\knut\dev
```
